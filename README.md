# ExamSystem
Celem projektu było stworzenie aplikacji konsolowej w języku C++, służącej do kompleksowego  zarządzania procesem egzaminowania. System umożliwia obsługę trzech różnych ról  użytkowników: Administratora, Wykładowcy oraz Studenta, zaimplementowanych przy użyciu  polimorfizmu i dziedziczenia.
